# Netflix Media Dataset EDA

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix dataset to understand content distribution, growth trends, genres, ratings, and runtime patterns.

## Objectives
- Analyze Movies vs TV Shows
- Study yearly content growth
- Identify top genres and countries
- Visualize rating distribution
- Analyze movie runtime trends

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Movies dominate over TV Shows
- Major growth observed after 2015
- Drama is the most common genre
- Most movies are 90-120 minutes long

## Conclusion
Netflix content has grown significantly in recent years with a strong focus on movies and drama genres.
